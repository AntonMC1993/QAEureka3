Formato de Pruebas de Software

A continuación, mostraremos un ejemplo de un formato de registro de Prueba Automatizada realizado en TestLink para la aplicación web desarrollada por Nicole Opazo; Desarrolladora Full Stack. Como se puede observar en las imágenes, creamos una carpeta con el Plan de Pruebas y subcarpetas con Suites de Prueba. Cada Suite de prueba contiene diferentes casos a probar. Por ejemplo, en la segunda imagen podemos apreciar que hemos establecido ciertas precondiciones básicas del entorno y sistema a fin de no incorporar pasos innecesarios. Estas precondiciones consisten en los programas que deben estar ejecutados al momento de realizar el testing.
