# ''EL(LA) ANALISTA QA''

Éste proyecto consiste en un Página web sencilla, tipo HTML5 estática, con diseño de librería Bootstrap, en la que se responden
las interrogantes sobre los tipos de prueba de software, las diferencias entre los tipos de Analistas QA, y un modelo de 
Plan de Prueba de Software descargable en PDF, más ejemplos de Casos de Prueba realizados en TestLink.

## Instalación

1. Ingrese a la página web github.com, copie el link del repositorio 
2. Haga clic derecho en su escritorio y seleccione la opción 'GIT BASH HERE'
3. Escriba en la ventana de comandos ''git clone'' , presione la barra de espacio y pegue el link del repositorio.
4. Presione la tecla Enter.
5. Aparecerá la carpeta en su escritorio.

## Uso

1. Abra su editor de código y con el Plugin Live Server puede visualizar el proyecto.
